# AAKAR  
## AI Assistive Kit for Autism

AAKAR is an **AI-powered assistive application** built using Flutter, designed to support **autistic individuals** in recognizing human emotions and managing emotional responses, while also helping **parents and teachers** better understand autistic behavior through symptom-based insights and tools.

The platform combines **emotion recognition, assistive analytics, calming mechanisms, and interactive games** to create a supportive, inclusive, and easy-to-use digital environment.
